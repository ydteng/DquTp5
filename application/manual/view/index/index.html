1.公共查询类

参数值
字段 释义
无 无
返回值
字段 释义
id banner的id号
url 图片路径

  • 获取省份（方法：get）

https://dqu.com/api/v1/province

参数值
字段 释义
无 无
返回值
字段 释义
id 省份id
name 省份名字

  • 获取学校（方法：get）

https://dqu.com/api/v1/school/:id

说明：根据省份id获取对应省份的学校

参数值
字段 释义
id 省份id
返回值
字段 释义
id 学校id
name 学校名称

provinceId

 学校所属省份id
level 学校等级
city 学校所在城市

 

 

2.登录注册类

  • 获取Token（方法：post）

https://dqu.com/api/v1/token/user

说明：code参数放在body中传递

参数值
字段 释义
code 微信小程序code
返回值
字段 释义
token 服务器生成的令牌

  • Token合法性验证（方法：post）

https://dqu.com/api/v1/token/verify

参数值
字段 释义
token 需要验证的token
返回值
字段 释义
isValid 结果（true或者是false）

 

 

3.用户信息类

  • 添加个人信息及收货地址（方法：post）

https://dqu.com/api/v1/address

说明：token在header中传入，其他在body中传入，json格式。

参数值
字段 释义
token 令牌（在header中传入）
nickname 昵称（区别于微信名）
real_name 真实姓名（保留，默认自定义）
mobile  手机号码
province_id 省份ID
school_id 学校ID
 detail 详细地址
返回值
字段 释义
code http状态码
msg 成功返回“ok”（主要）
errorCode 错误码

  • 获取用户地址信息（方法：get）

https://dqu.com/api/v1/address

说明：区别于上面的添加地址，接口地址相同，但是方法不同。token还是在get中。

参数值
字段 释义
token 令牌（在header中传入）
返回值
字段 释义
id 标记性id，可不使用
user_id 标记性id，可不使用
nickname 昵称
real_name 真实姓名
mobile 电话号码
detail 详细地址
province name 省份名称
school name 学校名称

  • 获取个人订单信息（方法：get）

https://api.brainholepi.com/api/v1/order/:id

说明：获取个人订单列表，返回的事按时间顺序从降序排序（即从晚到早）。token在header中传入

参数值
字段 释义
id 是用来分页用的id，防止一次性加载数据太多，每页返回10个数据
返回值
字段 释义
id 订单id，可用于查看订单详情
status （重要）订单状态，代码表示：1等待接单-2已接单派送中-3失败（订单取消或派送失败等）-4订单完成
cost 订单费用
start_point 订单起点
end_point 订单终点信息
item_type 订单物品类型（如快递，餐饮等）
order_num 订单号

cost 订单费用

end_point 订单终点信息
